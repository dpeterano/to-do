# TO DO LIST

## Main Idea & Features :

- Basic CRUD for Task
- Basic CRUD for Task Container
- Being able to change in wich container the task is 

## Design Idea :

- Black and white theme
- 3 Containers per row on desktop
- 1 Container per row on mobile & other small device
